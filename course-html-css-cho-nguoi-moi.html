Mô tả chi tiết

Khóa học HTML & CSS dành cho ai?

Khóa học nền tảng dành cho người mới muốn dựng giao diện web đầu tiên bằng HTML và CSS.

Khóa học dành cho ai

  • Người chưa từng học web
  • Sinh viên muốn bắt đầu front-end
  • Người chuyển ngành sang lập trình
  • Người cần làm landing page cơ bản

Mục tiêu khóa học

  • Hiểu cấu trúc HTML và semantic
  • Biết dùng CSS để tạo kiểu và bố cục
  • Làm layout với Flexbox và Grid
  • Hoàn thiện landing page đầu tay
Sau khóa học, bạn có thể tự dựng một website tĩnh có bố cục rõ ràng và hiển thị tốt trên nhiều thiết bị.

Nội dung khóa học

Lộ trình 5 module chính

Mỗi module được thiết kế để bạn hiểu nhanh, thực hành ngay và đi từ nền tảng đến dự án thực tế.

  • Cấu trúc tài liệu
  • Thẻ phổ biến
  • Semantic HTML
  • Selector
  • Box model
  • Typography và màu sắc
  • Flexbox
  • CSS Grid
  • Căn chỉnh phần tử
  • Media queries
  • Hình ảnh linh hoạt
  • Mobile-first
  • Trang giới thiệu cá nhân
  • Landing page sản phẩm
  • Hoàn thiện portfolio
